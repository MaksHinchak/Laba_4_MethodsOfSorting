Sorting Algorithms

This repository contains implementations of several classic sorting algorithms written in C++.

Implemented Algorithms
Bubble Sort

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order.
The algorithm continues until the array is fully sorted.

Time Complexity

Best case: O(n)
Average case: O(n²)
Worst case: O(n²)

Hoare Sort (Quick Sort – Hoare Partition Scheme)

This algorithm is based on the Quick Sort method using Hoare's partition scheme.
It selects a pivot element and divides the array into two parts: elements smaller than the pivot and elements greater than the pivot.

Time Complexity

Best case: O(n log n)
Average case: O(n log n)
Worst case: O(n²)

Shell Sort

Shell Sort is an improved version of insertion sort.
It allows the exchange of elements that are far apart by using a gap sequence.

Time Complexity

Best case: O(n log n)
Average case: depends on gap sequence
Worst case: O(n²)

Selection Sort

Selection Sort works by repeatedly selecting the smallest element from the unsorted part of the array and placing it at the beginning.

Time Complexity

Best case: O(n²)
Average case: O(n²)
Worst case: O(n²)
